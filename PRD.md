# Inner-Heart-AI-Chat PRD

## 1. 제품 개요

### 1.1 제품명
Inner-Heart-AI-Chat

### 1.2 목적
사용자가 마음속에 담아둔 감정을 안전하게 표현하고, 페르소나 기반 공감 응답을 받을 수 있는 웹앱을 제공한다.

### 1.3 대상 사용자
- 감정 정리가 필요한 일반 사용자
- 관계 대화 연습이 필요한 사용자
- 공감형 AI 응답을 원하는 사용자

## 2. 핵심 가치

- 감정 분석에 기반한 맞춤형 응답
- 관계/말투/강도별 페르소나 설정
- Google Drive 기반 RAG 지식 활용
- 대화 기록 자동 저장

## 3. 사용자 시나리오

### 3.1 페르소나 설정
사용자는 상대 관계, 감정, 원하는 반응, 강도, 말투를 설정한다.

### 3.2 메시지 전송
사용자는 메시지를 입력하고, 시스템은 감정 분석과 RAG 검색을 거쳐 응답한다.

### 3.3 기록 저장
대화 내용은 Google Sheets와 PropertiesService에 저장된다.

## 4. 기능 요구사항

### 4.1 웹앱 진입
- `doGet()`으로 채팅 UI를 제공한다.

### 4.2 페르소나 관리
- 관계, 감정, 원하는 반응, 강도, 말투를 저장한다.
- 세션 단위로 페르소나를 관리한다.

### 4.3 감정 분석
- 사용자 메시지를 감정/관계/강도로 분류한다.
- 핵심 키워드를 최대 3개 추출한다.

### 4.4 RAG 활용
- Google Drive 문서 6개를 불러온다.
- `###` 기준 섹션을 추출해 응답 전략에 반영한다.

### 4.5 최종 응답 생성
- 페르소나와 RAG 컨텍스트를 합쳐 Gemini 응답을 생성한다.
- RAG가 비어 있으면 기본 공감 응답으로 폴백한다.

### 4.6 데이터 저장
- 대화 기록을 Google Sheets에 저장한다.
- 분석 결과의 감정과 키워드도 함께 저장한다.

## 5. 비기능 요구사항

### 5.1 보안
- API 키는 코드에 직접 노출하지 않는다.
- 개인 설정 값은 PropertiesService에 저장한다.
- `.env`, `.clasp.json` 같은 로컬 파일은 Git에서 제외한다.

### 5.2 성능
- 캐시를 활용해 Drive 문서 반복 접근을 줄인다.
- 응답은 가능한 한 짧고 자연스럽게 유지한다.

### 5.3 사용성
- 카카오톡 스타일의 친숙한 채팅 UI를 제공한다.
- 모바일 화면에서도 읽기 쉬운 구성을 유지한다.
