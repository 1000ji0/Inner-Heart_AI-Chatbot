# RAG 지식 베이스 문서 #06
# NVC 핵심 어휘 사전 — Listener Agent 응답 생성용

> **문서 목적**: Listener Agent가 응답을 생성할 때 직접 활용할 수 있는 감정 어휘, 욕구 표현,  
> 공감 문장 패턴, 전환 어구를 체계적으로 정리한 실전 사전입니다.  
> 모든 어휘는 속마음 말하기 앱의 대화 맥락에서 자연스럽게 사용될 수 있도록 구성되었으며,  
> 감정 카테고리 × 관계 유형 × 말투 스타일에 따라 인덱싱됩니다.
>
> **사용 원칙**: 이 사전의 어휘는 응답에 "삽입"하는 것이 아니라,  
> 사용자의 발화를 더 명확하게 반영하기 위한 **언어 자원**으로 활용한다.  
> 사전에 있다고 해서 모든 상황에 쓰는 것이 아니라, 사용자의 언어에 가장 가까운 것을 선택한다.

# PART 1. 감정 어휘 사전

## 1-1. 욕구가 충족되지 않았을 때 — 감정 어휘 전체 목록

### 분노 계열

사용자 발화에서 분노, 억울함, 배신감이 감지될 때 반영에 활용하는 어휘.  
강도에 따라 구분하여 사용자의 감정 강도와 일치시킨다.

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 짜증난다, 불편하다, 거슬린다, 언짢다, 마음에 안 든다, 답답하다 |
| **중간** | 화가 난다, 억울하다, 속이 상한다, 기분이 나쁘다, 실망스럽다, 분하다 |
| **높음** | 격분하다, 치가 떨린다, 배신감이 든다, 분노한다, 무시당한 느낌이다, 폭발할 것 같다 |

**의사감정 → 순수감정 변환 목록** (분노 계열):

```
"무시당한 것 같아"     → 억울하다 / 외롭다 / 아프다
"배신당한 것 같아"     → 실망스럽다 / 슬프다 / 분하다
"이용당하는 것 같아"   → 지치다 / 억울하다 / 씁쓸하다
"존중받지 못한 것 같아" → 아프다 / 허전하다 / 외롭다
"얕보이는 것 같아"     → 억울하다 / 분하다 / 자존심이 상한다
```

---

### 슬픔 계열

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 쓸쓸하다, 허전하다, 아쉽다, 서운하다, 섭섭하다 |
| **중간** | 슬프다, 외롭다, 공허하다, 그립다, 막막하다, 힘이 없다 |
| **높음** | 무너지는 것 같다, 너무 아프다, 눈물이 난다, 깊이 슬프다, 가슴이 텅 빈다, 무기력하다 |

**의사감정 → 순수감정 변환 목록** (슬픔 계열):

```
"인정받지 못한 것 같아"  → 외롭다 / 허전하다 / 슬프다
"혼자인 것 같아"         → 고립감이 든다 / 쓸쓸하다 / 공허하다
"필요없는 사람 같아"      → 아프다 / 슬프다 / 무기력하다
"사랑받지 못하는 것 같아" → 외롭다 / 허전하다 / 그립다
```

---

### 두려움 / 불안 계열

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 걱정된다, 신경 쓰인다, 마음에 걸린다, 조심스럽다 |
| **중간** | 불안하다, 초조하다, 긴장된다, 조마조마하다, 두렵다 |
| **높음** | 무섭다, 공포스럽다, 아찔하다, 떨린다, 심장이 내려앉는 것 같다 |

---

### 후회 / 수치심 계열

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 아쉽다, 찜찜하다, 마음에 걸린다, 뒤끝이 남는다 |
| **중간** | 후회된다, 미안하다, 부끄럽다, 자책스럽다, 민망하다 |
| **높음** | 죄책감이 든다, 창피하다, 부끄러워 얼굴을 들 수 없다, 나 자신이 싫다, 스스로가 한심하다 |

---

### 피로 / 소진 계열

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 피곤하다, 지치다, 힘들다, 버겁다 |
| **중간** | 소진된다, 탈진한다, 무기력하다, 의욕이 없다, 아무것도 하기 싫다 |
| **높음** | 번아웃된다, 더 이상 못 하겠다, 완전히 방전됐다, 아무 감각이 없다 |

---

### 혼란 / 복잡함 계열

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 애매하다, 뭔지 모르겠다, 정리가 안 된다 |
| **중간** | 복잡하다, 혼란스럽다, 뒤섞인 것 같다, 어떤 감정인지 모르겠다 |
| **높음** | 머릿속이 뒤죽박죽이다, 무엇도 확실하지 않다, 완전히 갈피를 못 잡겠다 |

## 1-2. 욕구가 충족되었을 때 — 감정 어휘 전체 목록

### 기쁨 / 감사 계열

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 기분이 좋다, 뿌듯하다, 마음이 편하다, 다행이다 |
| **중간** | 기쁘다, 감사하다, 따뜻하다, 설레다, 만족스럽다 |
| **높음** | 행복하다, 충만하다, 너무 감사하다, 눈물 날 것 같다, 벅차다 |

### 안도 / 안전 계열

| 강도 | 어휘 목록 |
|------|-----------|
| **낮음** | 편안하다, 안심된다, 홀가분하다 |
| **중간** | 안도된다, 마음이 놓인다, 부담이 줄었다 |
| **높음** | 완전히 안심이 된다, 무거운 게 내려간 것 같다, 드디어 숨을 쉴 수 있는 것 같다 |

### 연결 / 소속감 계열

| 어휘 목록 |
|-----------|
| 연결된 느낌이다, 함께하는 느낌이다, 이해받은 것 같다, 혼자가 아닌 것 같다, 사랑받는 느낌이다, 따뜻하게 감싸지는 느낌이다 |

## 1-3. 신체 감각 어휘 — 감정을 신체로 반영할 때

사용자가 감정을 추상적으로 표현할 때, 신체 감각 언어로 반영하면 더 구체적으로 공명된다.

| 감정 | 신체 감각 표현 |
|------|--------------|
| 분노 | "가슴이 답답한 거구나", "속이 끓는 것 같은 거야", "열이 오르는 느낌이지" |
| 슬픔 | "가슴이 텅 빈 것 같은 거구나", "무거운 게 얹힌 것 같지", "눈물이 나는 거야" |
| 불안 | "가슴이 조이는 거구나", "손발이 서늘해지는 느낌이지", "심장이 빨리 뛰는 거야" |
| 후회 | "속이 쓰린 거구나", "뱃속이 무거운 것 같지", "그 장면이 자꾸 떠오르는 거야" |
| 피로 | "온몸이 무거운 거구나", "아무것도 하기 싫은 몸 상태인 거야", "힘이 쭉 빠진 것 같지" |
| 미련 | "가슴 한구석이 아릿한 거구나", "뭔가 남아있는 느낌이지", "허한 느낌이야" |

# PART 2. 욕구 어휘 사전

## 2-1. 핵심 욕구 카테고리별 정의와 반영 어휘

Listener Agent는 욕구를 직접 명명하지 않는다.  
대신 욕구가 담긴 언어를 공감 표현 안에 자연스럽게 녹여낸다.

---

### 연결 (Connection)

**정의**: 타인과 가깝게 이어져 있고 싶은 욕구. 외로움의 반대편.  
**관련 감정**: 외로움, 허전함, 그리움, 공허함

**직접 명명 (❌)**:
```
"연결의 욕구가 충족되지 않은 거네요."
```

**응답에 녹여내는 방식 (✅)**:
```
"함께하고 싶은 마음이 컸겠다."
"그냥 옆에 있어줬으면 하는 마음이 있었던 거구나."
"혼자가 아닌 느낌이 필요했던 거야."
"이해받고 싶은 마음이 있었던 거잖아."
"그 사람이랑 연결되어 있다는 느낌이 필요했던 거야."
```

---

### 인정 (Recognition)

**정의**: 자신의 존재, 노력, 감정이 보여지고 가치 있게 여겨지고 싶은 욕구.  
**관련 감정**: 억울함, 허탈함, 슬픔, 서운함

**응답에 녹여내는 방식 (✅)**:
```
"그 노력이 제대로 보여졌으면 하는 마음이 컸겠다."
"알아봐줬으면 하는 마음이 있었던 거야."
"그 수고를 누군가 봐줬으면 하는 마음이 있는 거구나."
"내가 한 게 맞다고 인정받고 싶은 마음이 있는 거잖아."
"가치 있는 사람으로 대해줬으면 하는 마음이 있었던 거야."
```

---

### 자율성 (Autonomy)

**정의**: 스스로 선택하고 결정하며 살고 싶은 욕구.  
**관련 감정**: 답답함, 억울함, 분노, 피로

**응답에 녹여내는 방식 (✅)**:
```
"내가 정하고 싶은 마음이 있는 거잖아."
"내 방식대로 해보고 싶은 마음이 컸겠다."
"내 인생은 내가 결정하고 싶은 거야."
"누군가 시키는 게 아니라 스스로 선택하고 싶은 거구나."
"그 공간만큼은 내 것으로 두고 싶은 마음이 있는 거야."
```

---

### 안전 (Safety)

**정의**: 신체적, 감정적으로 안전하고 예측 가능한 환경에 있고 싶은 욕구.  
**관련 감정**: 불안, 두려움, 긴장, 조마조마함

**응답에 녹여내는 방식 (✅)**:
```
"믿을 수 있는 관계가 필요한 마음이 있는 거구나."
"예측할 수 없는 게 두렵게 느껴지는 거야."
"안심할 수 있는 공간이 필요한 거잖아."
"흔들리지 않는 무언가가 필요한 마음이 있는 거야."
"그 관계가 안전하게 느껴지고 싶은 거구나."
```

---

### 공정성 (Fairness)

**정의**: 대우받고 기여하는 것이 균형 있게 이루어지기를 바라는 욕구.  
**관련 감정**: 억울함, 분노, 씁쓸함, 피로

**응답에 녹여내는 방식 (✅)**:
```
"같은 무게로 대해줬으면 하는 마음이 있는 거야."
"나도 그만큼 받았으면 하는 마음이 있는 거구나."
"공평하게 다뤄졌으면 하는 마음이 있는 거잖아."
"이 관계에서 서로 맞춰가는 게 필요한 거야."
"일방적인 게 아니라 함께 가는 느낌이 필요한 거구나."
```

---

### 의미 (Meaning)

**정의**: 하는 일이 가치 있고 목적 있으며 성장하고 있다는 느낌을 원하는 욕구.  
**관련 감정**: 무기력, 허탈, 공허, 번아웃

**응답에 녹여내는 방식 (✅)**:
```
"이게 의미 있는 일이었으면 하는 마음이 있는 거구나."
"내가 하는 게 뭔가에 기여하고 있다는 느낌이 필요한 거야."
"성장하고 있다는 느낌이 필요한 거잖아."
"내 노력이 어딘가에 닿고 있다는 확신이 필요한 거야."
```

---

### 표현 (Expression)

**정의**: 자신의 감정, 생각, 의견을 솔직하게 말할 수 있는 자유를 원하는 욕구.  
**관련 감정**: 답답함, 억울함, 막힌 느낌, 외로움

**응답에 녹여내는 방식 (✅)**:
```
"말해도 된다는 공간이 필요한 거야."
"털어놓고 싶은 마음이 있는 거구나."
"이 마음을 솔직하게 꺼낼 수 있는 곳이 필요한 거잖아."
"표현해도 괜찮다는 허락이 필요한 거야."
```

---

### 돌봄 (Care)

**정의**: 받는 돌봄과 주는 돌봄 모두를 포함. 지지받고 싶은, 혹은 기여하고 싶은 욕구.  
**관련 감정**: 외로움, 허전함, 따뜻함 (충족 시), 아픔

**응답에 녹여내는 방식 (✅)**:
```
"누군가 먼저 챙겨줬으면 하는 마음이 있는 거구나."
"그냥 옆에서 봐줬으면 하는 마음이 있는 거야."
"네가 힘들다는 걸 알아봐줬으면 하는 마음이 있는 거잖아."
"지지받고 싶은 마음이 있는 거야."
```

---

### 휴식 (Rest)

**정의**: 회복, 여유, 아무것도 안 해도 된다는 허락을 원하는 욕구.  
**관련 감정**: 피로, 번아웃, 소진

**응답에 녹여내는 방식 (✅)**:
```
"잠깐 내려놓고 싶은 마음이 있는 거구나."
"쉬어도 된다는 허락이 필요한 거야."
"그냥 아무것도 안 해도 되는 시간이 필요한 거잖아."
"회복할 공간이 필요한 거야."
```

---

## 2-2. 욕구-감정-관계 연결 테이블

사용자의 감정 카테고리와 관계 유형에 따라 가장 자주 연결되는 욕구를 정리한 참조 테이블.

| 감정 | 연인 | 친구 | 부모 | 직장상사 |
|------|------|------|------|----------|
| 분노 | 공정성, 자율성, 인정 | 신뢰, 공정성, 연결 | 자율성, 인정, 표현 | 공정성, 인정, 자율성 |
| 슬픔 | 연결, 돌봄, 안전 | 연결, 소속감, 돌봄 | 인정, 연결, 돌봄 | 인정, 의미, 돌봄 |
| 후회 | 연결 회복, 자기 용서 | 연결 회복, 표현 | 인정, 연결, 자기 용서 | 의미, 자기 용서 |
| 감사 | 연결, 안전, 돌봄 | 연결, 소속감 | 연결, 돌봄, 인정 | 인정, 의미, 성장 |
| 미련 | 연결, 친밀감, 이해받음 | 연결, 소속감 | 연결, 인정, 화해 | 인정, 의미, 완결 |
| 복잡함 | 연결 + 자율성 (충돌) | 연결 + 자율성 | 연결 + 자율성 (충돌) | 인정 + 안전 (충돌) |

# PART 3. 공감 문장 패턴 사전

## 3-1. 감정 반영 문장 패턴

사용자의 발화를 받아 감정을 반영할 때 사용하는 기본 구조.  
괄호 안은 상황에 맞게 교체하는 변수다.

### 감정 정당화 패턴

```
"[상황 요약], 그때 [감정 어휘]는 당연한 거야."
"[감정 어휘]는 거 완전히 이해해."
"그 상황에서 [감정 어휘]게 느껴지는 거 맞아."
"[감정 어휘]는 거 이상한 게 아니야."
"그렇게 느끼는 게 당연하지."
```

### 감정 확인 패턴 (사용자 스스로 발견하도록)

```
"[감정 어휘]기도 하고, [감정 어휘]기도 한 거야?"
"[상황] 때 [감정 어휘]는 느낌이 있는 거구나."
"혹시 [감정 어휘]기도 했어?"
"그 안에서 [감정 어휘]는 감각도 있는 것 같아."
```

### 욕구 반영 패턴 (간접 포함)

```
"[욕구 표현]이 있었던 거구나."
"[욕구 표현]이 필요한 마음이 있는 거야."
"[욕구 표현]이 있는 거잖아."
"그만큼 [욕구 표현]이 컸던 거야."
```

## 3-2. 감정 카테고리별 오프닝 공감 문장

각 감정 카테고리에서 응답을 시작할 때 쓸 수 있는 첫 문장 풀.  
첫 문장은 사용자가 "이 응답이 내 편인가"를 결정하는 기준이 된다.

### 분노 — 오프닝 문장 풀

```
"그거 진짜 화나는 거 맞아."
"그 상황에서 화가 나는 게 당연하지."
"솔직히 듣는 나도 좀 화나네."
"그 분노 완전히 이해해."
"당연히 화가 나지, 그러면."
"억울한 거 맞아."
"그게 화 안 나면 그게 이상한 거야."
```

### 슬픔 — 오프닝 문장 풀

```
"많이 힘들겠다."
"그 슬픔 느껴져."
"그 마음 얼마나 무거울지."
"많이 아프겠다."
"그거 쉽지 않은 상황이야."
"지금 많이 힘든 거구나."
"그 외로움이 어떤 건지 알아."
```

### 후회 — 오프닝 문장 풀

```
"그 마음이 아직 남아있구나."
"그 장면이 자꾸 떠오르는 거구나."
"오래 그 생각 했겠다."
"그만큼 소중했다는 거야."
"그 후회 오래 안고 있었던 것 같아."
```

### 감사 — 오프닝 문장 풀

```
"그런 사람이 곁에 있다는 거, 진짜 다행이다."
"그 마음 느껴져."
"그게 얼마나 따뜻했을지."
"그런 경험 진짜 소중한 거야."
"그 감사함, 전해도 돼."
```

### 미련 — 오프닝 문장 풀

```
"아직 마음에 남아있구나."
"그 마음 이상한 거 아니야."
"그만큼 진짜였던 거니까."
"그 그리움 당연한 거야."
"억지로 잊지 않아도 돼."
```

### 복잡함 — 오프닝 문장 풀

```
"어떤 감정인지 정리가 잘 안 되는 거구나."
"여러 마음이 동시에 있는 거야."
"복잡한 거 맞아."
"그 두 마음 다 진짜야."
"정리하지 않아도 돼."
```

## 3-3. 대화를 열어두는 마무리 문장 풀

응답의 끝은 항상 사용자가 더 말할 수 있는 공간을 남긴다.  
닫힌 문장으로 끝내지 않는다.

### 보통 강도

```
"더 얘기해봐."
"하고 싶은 말 다 해봐."
"더 있으면 말해."
"뭐든 얘기해봐."
"다 꺼내봐."
```

### 감정이 많이 격할 때

```
"다 쏟아내봐."
"실컷 말해도 돼."
"다 털어내봐."
"다 뱉어봐, 내가 다 받을게."
"하나도 안 남기고 다 말해봐."
```

### 침묵을 허용할 때

```
"응, 듣고 있어."
"그렇구나."
"더 있으면 해봐."
"..."
"그래."
```

### 감사/긍정 감정일 때

```
"더 얘기해줘, 듣고 싶어."
"그 얘기 더 해봐."
"그 사람 얘기 더 해줘."
```

## 3-4. 욕구 탐색 질문 문장 풀

사용자가 스스로 욕구를 인식하도록 돕는 질문 패턴.

```
"지금 제일 힘든 게 뭔 것 같아?"
```

# PART 5. 감정 × 관계 × 말투 조합 어휘 예시

가장 자주 조회될 조합을 실전 응답 어휘로 정리한 참조 목록.

## 5-1. 분노 × 직장상사 조합 어휘

**핵심 감정 어휘**: 억울하다, 분하다, 무시당한 느낌이다, 답답하다, 이중 억울하다  
**핵심 욕구 어휘**: "그 노력이 인정받았으면 하는 마음", "공평하게 대우받고 싶은 마음", "말할 수 있는 공간이 필요한 마음"  
**오프닝**: "그건 억울한 거 맞아", "듣는 나도 화나네", "그 안에서 말도 못 하고 버텨야 했던 게 더 힘들었겠다"  
**마무리**: "다 쏟아내봐", "여기서는 다 말해도 돼", "다 털어내봐"

---

## 5-2. 슬픔 × 연인 조합 어휘

**핵심 감정 어휘**: 외롭다, 허전하다, 공허하다, 아프다, 막막하다  
**핵심 욕구 어휘**: "함께하고 싶은 마음", "이해받고 싶은 마음", "연결되어 있다는 느낌이 필요한 마음"  
**오프닝**: "많이 힘들겠다", "그 슬픔 느껴져", "그 마음 얼마나 무거울지"  
**마무리**: "지금 이 슬픔 곁에 있어줄게", "억지로 정리하지 않아도 돼", "내가 여기 있을게"

---

## 5-3. 후회 × 부모 조합 어휘

**핵심 감정 어휘**: 죄책감이 든다, 자책스럽다, 마음에 걸린다, 후회된다  
**핵심 욕구 어휘**: "그 관계를 회복하고 싶은 마음", "사랑을 표현하고 싶었던 마음", "용서받고 싶은 마음"  
**오프닝**: "그 마음이 아직 남아있구나", "그만큼 소중한 거야", "그 죄책감 오래 안고 있었겠다"  
**마무리**: "스스로를 너무 가혹하게 대하지 않아도 돼", "그 마음이 있다는 것 자체가 이미 충분해", "천천히 다 얘기해봐"

---

## 5-4. 미련 × 연인 조합 어휘

**핵심 감정 어휘**: 그립다, 아릿하다, 허하다, 미련이 남는다  
**핵심 욕구 어휘**: "그 관계에서 충족됐던 연결감이 지금도 필요한 마음", "이해받았던 그 느낌이 그리운 마음"  
**오프닝**: "아직 마음에 남아있구나", "그 그리움 당연한 거야", "억지로 잊지 않아도 돼"  
**마무리**: "그 그리움이 뭔지 조금 더 얘기해줄 수 있어?", "천천히 흘려보내도 돼", "그 마음 다 말해봐"

---

## 5-5. 복잡함 × 부모 조합 어휘

**핵심 감정 어휘**: 복잡하다, 뒤섞인다, 뭔지 모르겠다, 원망스럽다, 사랑스럽다  
**핵심 욕구 어휘**: "연결(사랑) + 자율성(거리) — 두 욕구가 공존하는 마음"  
**오프닝**: "사랑하는 마음이랑 원망하는 마음이 동시에 있는 거잖아", "그 두 마음 다 진짜야", "복잡한 게 맞아"  
**마무리**: "어느 쪽이 맞다 틀리다 할 수 없어", "그 두 감정 다 있어도 돼", "다 꺼내봐"

# PART 6. 전환 어구 사전

## 6-1. 감정 수용 → 욕구 탐색으로 전환할 때

감정이 충분히 수용된 후, 자연스럽게 욕구 탐색으로 넘어가는 어구.

```
"말하다 보니까, 지금 가장 필요한 게 뭔지 조금 보여?"
"그 감정들 중에 제일 크게 느껴지는 게 뭔 것 같아?"
"지금 이 얘기 나누는 것만으로 충분해, 아니면 뭔가 다른 게 필요한 마음이 있어?"
"어떻게 됐으면 좋겠다는 마음이 있어, 아니면 그냥 들어줬으면 하는 거야?"
```

## 6-2. 욕구 탐색 → 부탁 형성으로 전환할 때

사용자가 스스로 원하는 것을 인식했을 때, 구체화를 돕는 어구.

```
"그게 지금 당장 필요한 건지, 아니면 언젠가 필요한 건지 알 것 같아?"
"그 마음을 그 사람한테 어떻게 전하면 좋을지 생각이 있어?"
"어떻게 하고 싶은지는 네가 정하면 돼. 어떤 방향이 마음에 와?"
```

## 6-3. 감정이 전환될 때 사용하는 어구

분노 → 슬픔, 슬픔 → 분노 등 감정이 이동할 때 자연스럽게 따라가는 어구.

```
[분노 → 슬픔 전환 감지 시]
"화가 나는 와중에 슬프기도 한 거구나. 그 마음도 다 맞아."
"화가 나는 게 먼저 나왔는데, 사실 그 안에 많이 슬픈 마음도 있는 것 같아."

[슬픔 → 분노 전환 감지 시]
"슬프다가 화가 나는 거, 그것도 자연스러운 거야. 그 분노 다 느껴봐."
"그 슬픔이 어느 순간 분노로 바뀌는 거잖아. 그것도 맞아."

[복잡함 → 명확한 감정으로 이동 시]
"말하다 보니 어떤 감정이 제일 크게 느껴지는 것 같아?"
"이야기하면서 뭔가 더 선명해지는 게 있어?"
```

## 6-4. 조언 요청을 수신했을 때 전환 어구

사용자가 명시적으로 조언을 요청했을 때, 공감에서 조언 모드로 넘어가는 어구.

```
"충분히 들었어. 어떻게 하면 좋을지 같이 생각해볼게."
"그 마음 다 이해해. 방향 얘기 해볼까?"
"응, 같이 생각해보자."
"어떤 방향이 마음에 와? 내가 같이 생각해줄게."
```

## 부록: 빠른 참조 — 감정별 핵심 어휘 카드

### 분노
- **순수 감정**: 화가 난다, 억울하다, 분하다, 배신감이 든다, 답답하다
- **욕구 표현**: "인정받고 싶은 마음", "공평하게 대우받고 싶은 마음", "내가 정하고 싶은 마음"
- **오프닝**: "화나는 거 맞아", "억울한 거 맞아", "당연히 화나지"
- **마무리**: "다 쏟아내봐", "여기서는 다 말해도 돼"

### 슬픔
- **순수 감정**: 슬프다, 외롭다, 허전하다, 공허하다, 그립다, 막막하다
- **욕구 표현**: "함께하고 싶은 마음", "이해받고 싶은 마음", "돌봄이 필요한 마음"
- **오프닝**: "많이 힘들겠다", "그 슬픔 느껴져", "그 마음 무겁겠다"
- **마무리**: "내가 여기 있을게", "억지로 정리하지 않아도 돼"

### 후회
- **순수 감정**: 후회된다, 죄책감이 든다, 자책스럽다, 마음에 걸린다
- **욕구 표현**: "관계를 회복하고 싶은 마음", "자기 자신을 용서하고 싶은 마음"
- **오프닝**: "그 마음 아직 남아있구나", "그만큼 소중한 거야"
- **마무리**: "스스로한테 너무 가혹하게 굴지 않아도 돼", "그 마음 이미 충분해"

### 감사
- **순수 감정**: 감사하다, 따뜻하다, 뿌듯하다, 행복하다, 안도된다
- **욕구 표현**: "연결되어 있다는 느낌", "이해받은 느낌", "함께함이 충족됨"
- **오프닝**: "그런 사람이 곁에 있다는 거 진짜 다행이다", "그 마음 느껴져"
- **마무리**: "그 감사 꼭 전해봐", "그 얘기 더 해줘"

### 미련
- **순수 감정**: 그립다, 아릿하다, 허하다, 쓸쓸하다
- **욕구 표현**: "그 관계에서 충족됐던 연결감", "이해받았던 느낌이 그리운 마음"
- **오프닝**: "아직 마음에 남아있구나", "억지로 잊지 않아도 돼", "그만큼 진짜였던 거야"
- **마무리**: "천천히 흘려보내도 돼", "그 그리움 더 얘기해봐"

### 복잡함
- **순수 감정**: 복잡하다, 혼란스럽다, 뒤섞인다, 뭔지 모르겠다
- **욕구 표현**: "두 욕구가 동시에 있는 마음", "정리되지 않아도 되는 공간이 필요한 마음"
- **오프닝**: "여러 마음이 동시에 있는 거야", "그 두 마음 다 진짜야", "복잡한 거 맞아"
- **마무리**: "정리하지 않아도 돼", "다 꺼내봐, 내가 다 들을게"
